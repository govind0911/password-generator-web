# ⚡ Neon Password Generator (Web)

A **neon-themed password generator** built entirely with **HTML, CSS, and JavaScript**.  
Generate strong, secure passwords instantly and copy them to your clipboard.  
The neon/cyberpunk UI makes it visually appealing and fun to use! 🌌

---

## 🚀 Features
- Set password length
- Include/exclude uppercase, lowercase, numbers, and symbols
- Neon/cyberpunk themed interface
- Copy password to clipboard with one click
- Works directly in any modern browser
- Can be hosted live using **GitHub Pages**
